# Roadmap

HooperTTS is preparing for its first public release. The current focus is
stability, documentation, and predictable CLI behavior.

## Near Term

- Expand tests for planner and rhythm edge cases.
- Add more sample scripts for different narration profiles.
- Improve packaging metadata before publishing.
- Improve dataset evaluation reports and warning quality.
- Add optional visual benchmark reports.

## Later

- Explore richer profile validation.
- Add more pronunciation dictionaries.
- Improve language and locale handling.
- Evaluate integration points with TTS engines.

## Non-Goals For 0.1

- No external runtime dependencies.
- No hosted service.
- No model-specific audio generation.
